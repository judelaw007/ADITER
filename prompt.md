- Verify accuracy across multiple sources

### 3. CREATE CONTENT

**Structure with numbered sections (1, 1.1, 1.2 style):**

**Core Framework (20-25%)**
- Statutory basis and definitions (bold key terms on first use)
- Legal principles

**Technical Rules (30-35%)**
- Rates, thresholds, conditions, tests
- Exemptions, reliefs, elections
- Use tables for rates/thresholds
- Bullet points ONLY for statutory conditions

**Analytical Depth (20-25%)**
- How provisions interact
- Cross-border implications
- Anti-avoidance considerations
- Compliance requirements

**Application Methodology (10-15%)**
- Step-by-step approach based on past exam patterns
- Decision frameworks

**Worked Examples (15-20%)**
- **Create the number specified in CSV**
- Each must have: Fact pattern, complete technical analysis, calculations with full working, statutory references, conclusion

**Visual Elements (20% of content - MANDATORY)**
- Tables, flowcharts, diagrams, comparison charts
- Decision trees, calculation templates, process maps

**Writing Standards:**
- Define technical terms on first use (authoritative definitions)
- **Bold**: key terms, section numbers, rates, thresholds
- *Italicize*: case names
- Numbered paragraphs for analytical content
- Clear, professional prose focused on substance

**2025 Developments:**
- Integrate naturally if relevant to understanding framework
- Do NOT create separate "2025 Changes" section
- If no changes, don't mention

---

### 4. AVOID (Critical!)

- ✗ Unnecessary conclusions or policy commentary

### 6. CREATE PROFESSIONAL WORD DOCUMENT

**Before creating document, read**: `/mnt/skills/public/docx/SKILL.md`

**Professional Formatting Standards:**

**Page Setup:**
- A4, 2.5cm margins, Portrait, Page numbers bottom center

**Typography:**
- Body: Calibri/Arial 11pt, 1.15 line spacing
- Heading 1: 18pt Bold, Navy Blue (#1F4E78)
- Heading 2: 14pt Bold, Dark Grey (#44546A)
- Heading 3: 12pt Bold, Medium Grey (#7F7F7F)

**Header/Footer:**
- Header Left: "ADIT Module 3.04 - Energy Resources"
- Header Right: [Topic Name]
- Footer Center: Page number
- All 9pt, Light Grey

**Title Block:**
```
                ADIT Module 3.04
            Energy Resources Option

        [Topic Number] - [Topic Name]
        
        ─────────────────────────────
        
                            Professional Study Note
                            Examination Level: [1/2/3]
```

**Tables (Critical!):**
- Header row: Dark blue (#1F4E78) background, white bold text
- Alternate rows: Light grey (#F2F2F2) shading
- Borders: Medium weight outer, thin inner gridlines
- Cell padding: 0.1cm
- Left-align text, right-align numbers

**Worked Examples:**
Format each as:
```
═══════════════════════════════════════════
WORKED EXAMPLE [#]: [Title]
═══════════════════════════════════════════

Facts:
[Fact pattern]

Required:
[What to determine]

Analysis:
[Step-by-step with statutory references]

Calculation:
[Full working with labels]

Conclusion:
[Final answer with legislation applied]
═══════════════════════════════════════════
```

**Visual Elements:**
- Flowcharts: Text boxes, clear YES/NO, arrows, professional colors
- Calculations: Indentation for working, tables for complex calcs, highlight final answers (light yellow #FFFF99)
- Keep examples and tables together (no page breaks mid-content)

**Final Checks:**
- [ ] Table of Contents generated
- [ ] All headings use Heading Styles (not manual formatting)
- [ ] Tables professionally formatted
- [ ] No orphaned headings
- [ ] No spelling/grammar errors
- [ ] Color scheme consistent
